Ocultar elementos

En continuación a display none, ahora se van a probar otras formas de ocultar los elementos que hay en la página web.

Visibility

Esta propiedad tiene 3 valores posibles visible (propiedad por defecto) hidden (oculta el elemento) y collapse (se usa en tablas).

Visibility hidden

También hace que los elementos visualmente no se vean, la diferencia con none es que estos si van a ocupar el espacio en el que están.

Opacity

Esta propiedad, dependiendo de los valores que se le den también sirve para ocultar elementos visualmente, esto mediante números entre el 0 y el 1 o con porcentajes 0% al 100%, actúa como un Font weight para elementos.

Elemento 1 en un div con 25% de opacidad.
Elemento 2 en otro div con 0.5 de opacidad.
Elemento 3 en otro div sin opacidad.

Entonces para desaparecer el elemento, con esta propiedad debe tener 0 ya sea en número o en porcentaje.
