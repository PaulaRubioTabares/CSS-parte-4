Tamaño a los elementos

Para dar un tamaño a los elementos ya habíamos visto la propiedad de width, y este lo que hace es que no importa el tamaño de la ventana de visualización de la página, siempre tendrá ese tamaño.

Por ello existen otras propiedades que igual asignan un tamaño a los elementos, pero en caso de que la ventana de visualización cambie su tamaño, el tamaño de ese elemento va a cambiar desde o hasta el valor indicado.

Estas medidas sirven para colocar límites a los elementos, también para que se ajusten mejor a los tamaños de las ventanas de los dispositivos.

Max width

La primera propiedad de este apartado que se va a ver es max width que asigna un ancho máximo que va a tener un elemento.

Elemento 1 en un div con propiedad de Width: 80rem.
Elemento 2 en un div con propiedad de Max-width: 80rem.

Comparando estas dos propiedades si se reduce la ventana del navegador, se ve como el elemento con solo la propiedad de width sobresale incluso de la sección de este tema y crea una barra de scroll, en cambio, si se añade la propiedad de Max-width, no permite que elemento sobresalga de esa sección.

Min width

La siguiente propiedad a añadir es min width que asigna un ancho mínimo a un elemento, en este caso, si la ventana del navegador es más pequeña que ese elemento, se va a crear una barra de scroll para visualizarlo.

Elemento 1 en un div con propiedad de Width: 80rem.
Elemento 2 en un div con propiedad de Max-width: 80rem y min-width: 50rem.

Max height y Min height

Max height define un alto máximo a los elementos.

Min height asigna un alto mínimo a los elementos.

Elemento 1 en un div con propiedad de Height: 3rem.
Elemento 2 en un div con propiedad de Max-height: 7rem y min-height: 5rem.

En ambos casos, si la pantalla se reduce, y el contenido del elemento quede más grande que el alto especificado, este puede salirse de su contenedor y sobreponerse a los elementos que estén debajo.

Aunque estas últimas dos propiedades en la práctica casi no son usadas.

Position

En esta sección se aborda el tema de position, esta propiedad determina, según el valor dado la posición que va a tener el elemento en la página web
Esta propiedad tiene los siguientes valores:

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

Para indicar en donde va a quedar el elemento se usan las propiedades de:

  • left
  • Right
  • Top
  • Bottom

Y el valor que se le asigne a estas propiedades, será la distancia que se va a mover el objeto según el tipo de posición que tenga.

Para ver cómo se comportan los elementos con cada valor en position se crean 3 div y se aplicara efecto siempre al div de la mitad.

Transition

Con esta propiedad se logra que los elementos que tienen más de un estado, como por ejemplo un hover que cambia el color de fondo, ese efecto tengan un cambio menos brusco. , esta propiedad se usa junto a hover, para que el efecto se aplique cuando el mouse se coloque encima

Transition property

Refiriéndose a la propiedad que va a tener el cambio de estado, ya sea el tamaño, el color de letra o fondo, entre otros.

Se pueden colocar individualmente las propiedades a cambiar, ejemplo:
Transition property: width, padding, background-color

O se pueden declarar que todas las propiedades que cambian tengan transición:
Transition property: all

Caja a cambiar.

Y en transition se tienen los valores de: transition-property: all; width: 20rem; padding: 3rem; background-color; border: 2px solid black; border-radius: 15px; color; pero el cambio es muy brusco.

Transition duration

Va a ser el tiempo en que se realiza el cambio de estado.

Ya con estas dos propiedades se puede lograr el efecto de una transición más suave.

Caja a cambiar.

En este caso se cambian los atributos de: width, padding, background-color, border, border-radius, color cuando el mouse se coloca encima del elemento con la propiedad hover.

Y las propiedades en las transiciones son: transition-property: all; transition-duration: 3s; donde se selecciona a cambiar todas las propiedades y que ese cambio sea en 3 segundos.

Transition delay

Como su nombre indica, coloca un delay o retraso en los cambios de atributos.

Caja a cambiar.

En este caso se cambian las propiedades de: width, padding, background-color, border-radius, color.

Y en transition se tienen los valores de: transition-property: all; transition-duration: 3s; transition-delay: 2s; en este caso el cambio del elemento cuando el mouse se coloca encima tarda 2 segundos en suceder, y lo mismo sucede cuando se retira el mouse.

Box Shadow

Una propiedad que no he colocado en esta serie de páginas de practicas de css es la de box shadow, que como su nombre lo indica, coloca una sombra en la caja según la forma dada en el box model, contando con los siguientes valores.

Offset x

Este valor es el Primer valor que se declara en box shadow y representa el desplazamiento que tendrá la sombra en el eje x respecto al bloque original.

Offset Y

Este valor es el Segundo valor que se declara en box shadow y representa el desplazamiento que tendrá la sombra en el eje y.

Caja Con Sombra

Ya con estos dos valores se puede crear una "segunda caja" la cual será la sombra, en este caso la caja tiene la siguiente propiedad: box-shadow: 8px 8px; donde tiene un desplazamiento de 8 pixeles tanto en x como en y, copiando las propiedades del box model del elemento.

También se pueden dar valores negativos

Caja Con Sombra
